## Important Notes for Reviewers

- Nafaa is a B2B (business-to-business) POS application for shopkeepers
- All data collected is necessary for operating a point-of-sale system
- We do NOT sell user data
- We do NOT use data for third-party advertising
- Payment processing is handled by Stripe (PCI compliant)
- All data is encrypted in transit (HTTPS) and at rest
